Welcome to KOALA

  

Willkommen bei KOALA

 

 

 

KOALA is a web-based application, designed to help you managing your AWS compatible cloud infrastructures (IaaS). The Amazon AWS public cloud and private cloud infrastructures based of Eucalyptus or Nimbus are supported   KOALA ist eine internet-basierte Anwendung, um AWS-kompatible Cloud Computing Infrastrukturen (IaaS) zu steuern. Die Public Cloud Amazon AWS und Private Cloud Infrastrukturen mit Eucalyptus oder Nimbus werden unterstützt

 

 

 

KOALA itself is able to run inside the public cloud platform (PaaS) Google App Engine and the private cloud PaaS AppScale   KOALA selbst läuft innerhalb der Public Cloud Plattform (PaaS) Google App Engine und der Private Cloud PaaS AppScale

 

 

 

KOALA controls the Elastic Computing Cloud (EC2), helps interacting with Simple Storage Service (S3) and Elastic Block Store (EBS) storage resources and supports Elastic Load Balancing (ELB)   KOALA steuert die Elastic Computing Cloud (EC2) und unterstützt auch die Arbeit mit Simple Storage Service (S3) und Elastic Block Store (EBS) Speicher-Ressourcen und mit elastischer Lastverteilung via Elastic Load Balancing (ELB)

 

 

 

KOALA is not a marketplace for cloud resources. It just helps you to manage the cloud resources you have already access to   KOALA ist kein Marktplatz für Cloud-Ressourcen. Es ist ein Werkzeug um Cloud-Ressourcen zu bedienen, zu denen man bereits Zugriff hat

 

 

 

 

 

 

KOALA is Open Source and the usage is without cost   KOALA ist Open Source und die Benutzung ist kostenfrei
