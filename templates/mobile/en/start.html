Welcome to KOALA

 

KOALA is a web-based application, designed to help you working with your AWS compatible cloud services and infrastructures (IaaS). The Amazon AWS public cloud and private cloud services based of Eucalyptus, OpenNebula or Nimbus are supported. The stoage services Google Storage and Host Europe Cloud Storage can be used with KOALA too

 

KOALA itself is able to run inside the public cloud platforms (PaaS) Google App Engine and the private cloud PaaS AppScale and typhoonAE

 

The source code and documentation is available at the KOALA project site

 

Please send improvement suggestion, comments and bug reports to koalacloudmanager@googlemail.com
