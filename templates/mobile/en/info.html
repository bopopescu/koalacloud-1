Information about the KOALA Cloud Manager

 

The application allows to manage Cloud infrastructures (IaaS) that are compatible to Amazon Web Services (AWS). Besides Amazon AWS private cloud infrastructures based on Eucalyptus, OpenNebula and Nimbus are supported.

 

The source code and documentation is available at the KOALA project site.
